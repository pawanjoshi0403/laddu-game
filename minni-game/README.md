# minni game

A colourful Android Studio game for a 3-year-old child. It teaches:

- Color recognition
- Number counting from 1 to 10
- Alphabet matching from A to J
- Tap-based choice making
- Animal and food association

## Open in Android Studio

1. Open Android Studio.
2. Choose **Open**.
3. Select this folder:
   `/Users/pawanjoshi/Documents/Codex/2026-04-28/i-want-to-make-a-2d`
4. Let Gradle sync.
5. Run the app on an emulator or Android phone.

## How the Game Works

The app starts with a colourful opening screen where animals and birds move across the scene.
After that, the home page shows a fixed heading: `minni game`.

The home page has three large picture buttons:

- Colours
- Numbers
- Alphabets

## Features

- Baby-style spoken prompts using Android `TextToSpeech`.
- Numbers game starts with large monkey, elephant, and chicken cards.
- After choosing an animal, only that animal appears on its own feeding screen.
- Food counting game: banana, sugarcane, and peanuts with large cartoon-style icons.
- Tap food to count aloud from 1 to 10.
- Colours game with a big cartoon target and three side-by-side colour options.
- Wrong colour answers are spoken aloud and corrected.
- Alphabets game with large object icons like apple for A and three side-by-side letter options.
- Celebration popup with fireworks after 10 correct actions.
- Home button on the celebration popup.
- Gentle looping tone music with a mute button.
